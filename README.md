# Cosmic Beats 🎧🌌

A beautiful, free music streaming web app inspired by Spotify & Gaana, powered by **JioSaavn unofficial API** (320kbps streaming + downloads).

## Features
- Real-time song search (Bollywood, Punjabi, English hits)
- 320kbps high-quality playback
- Live cosmic audio visualizer (pulsing bars)
- AI "Surprise Me" random mix
- Mini player + full-screen player
- Favorites (saved in browser)
- Shuffle, repeat, queue, direct song download
- Dark cosmic theme with purple/pink gradients

## How to Run Locally
1. Download or clone the repo
2. Double-click `index.html`
3. Open in browser – search and play!
